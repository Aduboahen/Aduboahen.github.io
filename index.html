Featured Project: Plasmodium falciparum Variant Calling Pipeline

The Plasmodium falciparum Calling Pipeline is a Nextflow-based workflow designed to identify resistant genes in Plasmodium falciparum whole genome sequencing data

View the repository on GitHub

Usage Instructions:

To use this pipeline, follow these steps:

git clone https://github.com/Aduboahen/pfal_variant_calling.git
cd pfal_variant_calling
docker pull mugicrow/pfal_variant_call:latest
./scripts/test_run.sh

For a single sample run:

./scripts/start_docker.sh
nextflow run pf.resistance.nf --sampleid test --reads "./input/test_{1,2}.fastq.gz" --outDir ./output -resume

For batch processing:

./scripts/run_batch.sh

For detailed instructions and more information, please refer to the README.
